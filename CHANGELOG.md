# Changelog

All notable changes to Forge Map 3D will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and the project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Open-source project documentation and community health files.
- Offline geometry regression checks and a unified `npm run check` release gate.
- GitHub Actions CI for Node.js 22 and 24.

### Changed

- Forge Map 3D is now scoped to locations in the Netherlands only.
- Dutch locations use OpenStreetMap map layers with enhanced 3DBAG building
  geometry where available.
- Project licensing is now AGPL-3.0-only.

## [0.1.0] - Unreleased

Initial public preview.
